# Car Rental Web App

Simple car rental system built with Flask and MySQL.

## Tech Stack
- Python (Flask)
- MySQL
- HTML/CSS (Bootstrap)

## Setup
1. Create virtual environment
2. Install dependencies
   pip install -r requirements.txt
3. Configure environment variables
   cp .env.example .env
4. Initialize database
   python init_db.py
5. Run application
   flask run
